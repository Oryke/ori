# Code of Conduct

## Our Commitment

We are committed to making participation in the ORI community a welcoming, respectful, and harassment-free experience for everyone, regardless of background, identity, experience level, or personal characteristics.

We value collaboration, curiosity, constructive feedback, and learning.

## Expected Behaviour

Community members are expected to:

- Be respectful and considerate toward others.
- Communicate clearly and professionally.
- Welcome different perspectives and experiences.
- Provide constructive feedback.
- Focus on improving the project and supporting contributors.
- Respect the decisions of maintainers and project contributors.

## Unacceptable Behaviour

The following behaviours are not acceptable:

- Harassment, discrimination, or personal attacks.
- Trolling, insulting, or deliberately disruptive behaviour.
- Publishing others' private information without permission.
- Threats, intimidation, or abusive language.
- Any conduct that creates an unsafe or hostile environment.

## Reporting Issues

If you experience or witness unacceptable behaviour, please report it by opening a private communication channel with the project maintainers.

Reports will be reviewed and handled respectfully and confidentially.

## Enforcement

Project maintainers are responsible for clarifying and enforcing this Code of Conduct.

Actions may include:

- A private warning.
- Temporary restriction from participation.
- Removal from project spaces.
- Permanent banning from contributing.

## Scope

This Code of Conduct applies within all ORI community spaces, including:

- GitHub repositories and discussions.
- Pull requests and issues.
- Any official ORI community channels.

## Attribution

This Code of Conduct is adapted from the Contributor Covenant, a community standard for open-source projects.
