# The ORI Manifesto

Open source has grown faster than our ability to navigate it.

Repositories contain thousands of files,
hundreds of issues,
dozens of contributors,
and years of engineering decisions.

Finding where to begin should not be the hardest part.

ORI exists because repository data should become repository intelligence.

GitHub tells us what exists.

ORI helps us understand what matters.

We believe developers deserve guidance,
maintainers deserve clarity,
and open source deserves better navigation.

ORI is not another GitHub client.

ORI is an intelligence engine.

This is only the beginning.

ORI-ON! ✨
